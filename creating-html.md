# Creating Your First HTML File

Follow these steps to create a basic HTML file:

1. Open a text editor (e.g., Notepad, VS Code).
2. Type the following code:
   ```html
   <!DOCTYPE html>
   <html>
   <head>
       <title>My First Web Page</title>
   </head>
   <body>
       <h1>Hello, World!</h1>
   </body>
   </html>
